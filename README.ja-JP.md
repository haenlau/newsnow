# Air1 News

Air1 News is a real-time news reader in the Air1 Quick Tools collection.

This fork keeps the lightweight news aggregation experience and applies Air1 branding, Cloudflare deployment defaults, and a clearer product identity.

## Quick Start

```sh
corepack enable
pnpm install
pnpm dev
```

## Deployment

- Build command: `pnpm run build`
- Output directory: `dist/output/public`
- Suggested Cloudflare Pages project name: `air1-news`
- D1 binding name: `AIR1_NEWS_DB`
